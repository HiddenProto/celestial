What is Drew's Learning Academy?

Drew’s Learning Academy is a modern educational center focused on helping students build strong academic foundations. It offers a structured environment where learners can develop skills in reading, writing, math, and critical thinking.

How does Drew's Learning Academy teach students?

The academy uses small group sessions and personalized instruction to make sure every student gets the attention they need. Teachers adjust lessons based on each student’s pace, making the experience more effective and enjoyable.

Why is Drew's Learning Academy beneficial?

Beyond academics, Drew’s Learning Academy encourages good study habits and long-term growth. Students learn how to stay organized, manage time, and approach challenges with a positive mindset.

☁️ Cloudflare Mode Optimizes the proxy to better handle Cloudflare-protected sites.

Automatically switches transport to epoxy and proxy to Bumblcat RRC.

Transport, proxy, and wisp settings are locked while enabled to prevent incompatible combinations.

better compatibility with cloudflare-protected sites — locks transport & proxy to optimal settings

🔒 transport & proxy locked — using epoxy + bumblcat rrc for best cloudflare compatibility

Transport Controls how proxied requests are sent through the network.

epoxy — WebAssembly-based, better site compatibility. Slower. Type: client-side.

libcurl — C-based, faster and more stable. Recommended. Type: client-side.

change how the proxy is handled, can lead to sites that were broken to work or lead to sites that were working to break

Proxy Controls how page content is rewritten for proxying.

Bumblcat RRC — private fork of scramjet, faster + improved. Recommended. Type: client-side.

ultraviolet — older rewriter, works with some legacy sites. Type: client-side.

change which rewriter engine handles proxied pages

Wisp Server The backend WebSocket server that relays your proxy traffic. Required for the proxy to work.

bumblcat's server — private server hosted by bumblcat. Recommended default. Has full cross-origin support, meaning more apps and games load correctly. Note: slightly slower on first load due to server fetch & cross-origin handling. Type: private server.

mercury workshop — free public community server. Shared with other users. No cross-origin support. Type: public server.

custom — enter your own wisp server URL. Type: self-hosted server.

the server that routes your proxied traffic
